# executionflow

This repository is configured to deploy to GitHub Pages automatically from the `main` branch.

To publish your HTML page:
1. Add your HTML file to the repository as `index.html` in the repository root.
2. Push to `main`.
3. Open the **Actions** tab and wait for the **Deploy GitHub Pages** workflow to finish.

Your page will be available at:
`https://vincentk16.github.io/executionflow/`
